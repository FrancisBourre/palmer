Classpublic class Sequencer
InheritanceSequencer AbstractCommand Object
Implements MacroCommand, CommandListener

Sequencer object encapsulate a set of Commands to execute.

Sequencer is a first-in first-out stack (FIFO) where commands are executed in the order they were registered.

Default, the Event object received in the execute is passed to each commands contained in this sequencer. But you can relay event from command to command using useEventFlow property.

The Sequencer class extends AbstractCommand and so, dispatch an onCommandEnd event at the execution end of all commands.

useEventFlowproperty 
useEventFlow:Boolean

Indicates if Sequencer use command event flow or only main source event data.

If true next command is executed using last command event result. If false, only use the main source event data passed-in Sequencer.execute() call.

The default value is false.
