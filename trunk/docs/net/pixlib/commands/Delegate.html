Classpublic class Delegate
InheritanceDelegate AbstractCommand Object
Implements TickListener

The Delegate encapsulate a method call as an object. The Delegate class provides two ways to encapsulate a method call :
  • By creating a new Delegate instance you can wrap the call into a command object.
  • By calling the Delegate.create method you can get an anonymous function which will encapsulate the call.

onTick()method 
public function onTick(e:Event = null):void

Method called on each iteration according to TickBeacon time slicing approach.

The onTick method is very similar to the old onEnterFrame method, but doesn't specifically occurs when entering in a new frame of a frame-based animation. The tick could be the result of a setInterval call, or position change in a video's timecode...
