applyStyle()method 
public function applyStyle(styleName:String, field:TextField = null, beginIndex:int = -1, endIndex:int = -1):CSSStyleFormat

Applies passed-in styleName style to field TexField object.

Uses custom methods defined using addParser() to built TextFormat object.

A TextFormat object is automatically created and saved into cache system.

Parameters

styleName:String — Style name (in current StyleSheet) to apply
 
field:TextField (default = null) — Targeted TextField object
 
beginIndex:int (default = -1)
