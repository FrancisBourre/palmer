Classpublic class PreprocessorParser
InheritancePreprocessorParser XMLParser AbstractParserCommand AbstractCommand Object

The PreprocessorParser class allow xml context pre processing from xml context nodes.

Context preprocessing order is :

  • ApplicationLoader.addProcessingMethod() job
  • ApplicationLoader.addProcessor() job
  • ContextPreprocessorLoader job

See also

net.pixlib.ioc.context.processor.ContextProcessor

Method name that processor must implement to transform xml context data.

Used only when method attribute is not defined in preprocessor xml node.

The default value is process.
