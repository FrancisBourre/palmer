Packagenet.pixlib.log.layout
Classpublic class DeMonsterDebuggerLayout
InheritanceDeMonsterDebuggerLayout Object
Implements LogListener

The DeMonsterDebuggerLayout class provides a convenient way to output messages through DeMonsterDebugger application.

Just provide the DeMonsterDebugger Client class to the layout to allow communication.

To get more details, visit: http://www.demonsterdebugger.com/
