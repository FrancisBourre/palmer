public static function nextPassword(useNumber:Boolean = true, mixCap:Boolean = true, useOther:Boolean = true, min:Number, max:Number):String

Returns a new random generated password.

Parameters

useNumber:Boolean (default = true) — true to enable number in string ( default is true )
 
mixCap:Boolean (default = true) — true to enable lower and uppercase mix ( either just lowercase are used ). Default is true
 
useOther:Boolean (default = true) — true to enable special character use ( default is false )
 
min:Number (default = NaN) — ( optional )Minimum length for generated password, default is Random.DEFAULT_PASSWORD_LENGTH
 
max:Number (default = NaN) — ( optional )maximum length for generated password
